2. Privacy Policy

This is a plain-language privacy summary for Careme:

  • We store only the data needed to run the app, such as account information, saved recipes, and optional feedback.
  • We do not sell personal information.
  • If analytics are enabled, they are used to improve reliability and usability.
  • We work with trusted partners for sign-in, storage, and ai/large language models
  • You can request account data removal by opening a GitHub issue on the project repository.

This policy may be updated as features evolve. Last updated: February 16, 2026.

4. General FAQ

What is Careme?

Careme is a recipe planning app that uses store availability and seasonal ingredients to suggest practical weekly meals.

Do I need an account?

You can browse some pages without signing in, but an account is needed to save recipes, track favorites, and keep history.

Can I share a recipe page?

Yes. Every recipe page has its own shareable link.

How do I suggest improvements?

Use the GitHub page to share ideas, report problems, or ask questions.
